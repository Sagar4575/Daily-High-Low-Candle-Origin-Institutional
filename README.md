# Daily High Low – Candle Origin (Institutional)

A free TradingView indicator that plots the current and previous daily high and low levels, anchored to the exact candle where each level was formed.

This indicator is designed for intraday traders who use daily highs and lows as key reference levels for price action, liquidity, and market structure analysis.

---

## What This Indicator Does

- Plots current day high and low levels
- Plots previous day high and low levels
- Anchors each level to the candle that created it
- Updates current day levels dynamically
- Locks previous day levels (non-repainting)

---

## How to Use

1. Open TradingView
2. Open any intraday chart (5m, 15m, 1h, etc.)
3. Add the indicator to the chart
4. Use daily high and low levels as:
   - Support and resistance
   - Liquidity reference points
   - Intraday bias levels

---

## Best Use Cases

- Intraday trading
- Price action analysis
- Liquidity sweep identification
- Support and resistance mapping

Works on Forex, Crypto, Stocks, and Indices.

---

## Installation

### Option 1 (Recommended)
Use the published indicator directly on TradingView.

### Option 2 (Manual)
1. Copy the code from this repository
2. Paste it into the TradingView Pine Editor
3. Save and add to chart
